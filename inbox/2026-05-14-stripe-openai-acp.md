# Stripe × OpenAI — Instant Checkout in ChatGPT + Agentic Commerce Protocol(ACP)

## 要約
Stripeが2026年4月30日、ChatGPT内での「Instant Checkout」提供を発表。OpenAIと共同開発したAgentic Commerce Protocol(ACP)を同時公開。ACPはエージェントが商取引を安全に実行するための共通プロトコルで、Google AP2・Coinbase x402と並ぶ標準候補となる。Stripeは既存カード決済ネットワークとの互換性を重視した設計。

## なぜKYAに関係あるか
ChatGPTを入口とするエージェント決済のユーザー体験が確立されると、KYAが対応すべき「委任の粒度・取消し権・責任帰属」設計の要件が具体化する。ACP・AP2・x402の三標準競合の構図もKYA技術選択に影響。
