# SecureAuth、Agentic Authority Platform 正式リリース — AI エージェント ID をリアルタイム認可

## 要約
SecureAuth が 2026-05-19 に Agentic Authority Platform を発表。AI エージェントが実行する全 API コール・ツール呼び出し・トランザクションをゼロトラスト原則でリアルタイム認可・監査する制御レイヤー。業界調査では企業 AI エージェントの 91% が過剰特権状態にあり、78% には監査証跡がないと指摘。Mark van Oppen を CRO に採用しエンタープライズ展開を加速。

## なぜ KYA に関係あるか
KYA の「runtime 認可・監査」レイヤーの具体実装例。SecureAuth は 2026-04-29 に業界初の公開 Agent Trust Registry も開設しており、エージェント ID 登録インフラとして KYA フレームワーク設計の参照先になり得る。
