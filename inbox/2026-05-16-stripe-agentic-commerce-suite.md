# Stripe、「Agentic Commerce Suite」でエージェント向け決済インフラを整備

## 要約
Stripe は Agentic Commerce Suite（ACS）を発表。Shared Payment Tokens でエージェントが購買者の支払認証を安全に転送する仕組みを提供。URBN・Etsy・Ashley Furniture 等が採用。BigCommerce・commercetools 等 e コマース PF とも統合。Checkout Sessions API ベースで送料・税計算も統合処理。

## なぜ KYA に関係あるか
Shared Payment Token は「エージェントに付与された支払権限の委任」モデルであり、KYA のスコーピング（権限バインディング）と直接競合する設計。Stripe がエージェント認証レイヤーをどこまで取り込む意図かを要確認。
