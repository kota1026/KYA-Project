# arXiv 2603.25190「zk-X509: Privacy-Preserving On-Chain Identity from Legacy PKI via ZKP」

## 要約
RISC-V ベースの zkVM（SP1）を用い、既存 X.509 証明書のチェーン検証・失効確認・鍵所有証明をオンチェーンで秘匿実行。秘密鍵は ZK 回路に入らず OS キーチェーン（macOS Secure Enclave / Windows TPM）で処理。Groth16 オンチェーン検証コスト約 300,000 gas。Sybil 耐性のため nullifier を設定可能。

## なぜ KYA に関係あるか
既存 PKI（国家・企業 CA）を ZKP で橋渡しするアプローチは、KYA の Selective Disclosure 基盤として有力。エージェント ID をレガシー証明書から導出する実装パスを示す。
