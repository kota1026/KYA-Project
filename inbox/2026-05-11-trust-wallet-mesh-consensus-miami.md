# Trust Wallet・Mesh、Consensus Miami でエージェント向けウォレット再設計を発表

## 要約（3-5文）
Trust WalletはEIP-8004（エージェントのオンチェーンIdentity＋信用スコア）実装と開発者向けAgent Kitを発表。Meshはエージェント自律取引・送金・self-custody タスクを支援するAIウォレットビジョンを披露。両社は「AIラボが独自ウォレットを出す」と予測しXとGrokを例示。「エージェント行動の責任はデプロイした機関にある」との原則を強調。

## なぜ KYA に関係あるか
EIP-8004のオンチェーンAgent Identity概念はKYAの技術実装候補。競合として Track A に追加すべき2社。ノンカストディ型設計との比較検討にも有用。
