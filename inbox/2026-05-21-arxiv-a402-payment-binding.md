# arXiv:2603.01179 — A402: Binding Cryptocurrency Payments to Service Execution for Agentic Commerce

## 要約（3-5 文）
x402 プロトコルを拡張した A402 を提案する論文。自律エージェント固有の 3 課題（法的 ID の欠如・未知サービス提供者との取引・低遅延かつ細粒度の決済要求）に対処する。支払いとサービス実行を暗号技術でバインドし、二重支出と不完全履行をオンチェーンで防止。既存の x402 エコシステムと後方互換性を保ちつつ追加保証を提供する設計。

## なぜ KYA に関係あるか
x402 の最大の KYA ギャップである「エージェントの法的 ID 欠如」を正面から扱う学術提案。Track C の「ペイメントプロトコル安全性」テーマと直結し、KYA 権限バインディング設計の参照資料となる。SoK 論文（2604.15367）と合わせて読むと理解が深まる。
