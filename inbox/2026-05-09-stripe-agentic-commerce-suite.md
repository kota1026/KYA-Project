# Stripe Agentic Commerce Suite 発表 — SPT で Visa/Mastercard/BNPL 対応

## 要約
StripeがAgentic Commerce Suiteを発表。中核技術はShared Payment Token（SPT）: バイヤーの決済情報を実際に渡さずエージェントが代理決済できるスコープ付きトークン。Mastercard Agent Pay・Visa Intelligent Commerce・Affirm/Klarnaに対応拡大。Google Gemini・Meta・Microsoft Copilot・OpenAI ChatGPTとの販売チャネル統合も発表。既存決済スタックとの後方互換を維持しつつエージェント決済に対応。

## なぜ KYA に関係あるか
SPTは「ユーザー委任の証明をトークン化」する仕組みで、KYAの権限委任モデルとアーキテクチャ上の類似点が高い。非カストディ型KYAとの比較検討対象。Track A 競合マップ上でStripeの位置づけ確定に必要。
