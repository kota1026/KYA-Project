# The Agentic Web: M2M 決済プロトコル競争の内側（Emerging Fintech）

## 要約（3-5文）
Emerging Fintech の分析記事が、x402（Coinbase）・MPP（Stripe）・AP2（Google/Visa Mastercard）・A2A の4プロトコルが M2M 決済標準の座を争う構図を整理。x402 は HTTP ネイティブで開発者採用が先行し、MPP はカード/ACH/ステーブルコインのマルチレール対応が強み。AP2 はネットワーク各社の後ろ盾でエンタープライズ寄り。プロトコル収束まで 18-24ヶ月と推定。（推定）

## なぜ KYA に関係あるか
KYA ソリューションがどのプロトコルに乗るかは実装レイヤーに直結。プロトコル標準化の帰趨次第で、KYA のアイデンティティ・レイヤーの設計も影響を受ける。Track A の競合全社がどのプロトコルを採用しているか整理するための地図として活用可能。
