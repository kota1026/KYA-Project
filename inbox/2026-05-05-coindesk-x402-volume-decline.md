# CoinDesk：AI エージェントが Web 経済を破壊、x402 で対抗 — ただし取引量は 77% 減

## 要約
CoinDesk が x402 プロトコルの課題を報じた。調整後取引量は 2025-11 の $5.15M ピークから 2026-05 には $1.19M へ約 77% 急減。一方 Cloudflare ネットワーク全体では AI エージェントによる無料コンテンツ大量スクレイピングが深刻化しており、x402 による「従量課金」が有効な対抗策と主張。承認率の低さが普及の足かせとなっている。

## なぜ KYA に関係あるか
x402 の取引量推移は KYA ユースケース（エージェント決済）の市場規模を測る指標。取引量減少は市場普及の課題を示す重要シグナルで Track D（ビジネスモデル）分析にも影響。
