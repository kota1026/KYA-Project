# AI 間決済が 7 か月で 1 億件を突破—USDC / Base が事実上の標準に

## 要約

2025年10月から追跡開始した Agentic Payment の累計件数が 2026-05-25 時点で 1 億 1,310 万件・決済総額 1,630 万ドルに達した(1 件あたり平均 0.48 ドル)。Artemis データによると件数の 82% 以上が Coinbase の L2 ネットワーク Base 上で処理され、98.6% が USDC 建て。ステーブルコインの低手数料・24 時間リアルタイム決済が AI micropayment の事実上インフラとなっている。

## なぜ KYA に関係あるか

Track D の事業モデル検討において「USDC / Base がデファクトスタンダードになった」ことは市場前提として重要。KYA インフラが想定すべきトランザクション単価・規模感の基準値として使える。競合調査(Track A)でも Base エコシステム優位を前提にした分析が必要。
