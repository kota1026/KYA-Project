# Sumsub、エージェントを人間 ID にバインドする「Agent-to-Human Binding」を発表

## 要約
Sumsub は 2026-01-29、AI エージェント Verification 製品を公開。device intelligence・liveness 検証・mule network 検出を組み合わせ、すべてのエージェント行動を本人確認済み人間に紐付ける設計。2025 年に多段階協調攻撃が前年比 180% 増と報告（同社 Identity Fraud Report）。

## なぜ KYA に関係あるか
「エージェント → 人間 principal のバインディング」は KYA の中核概念と同一。Sumsub は KYC インフラ大手であり、既存 KYC 顧客へのバンドル販売が可能な強力な競合。Track A への記録が急務。
