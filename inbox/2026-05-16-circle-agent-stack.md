# Circle、AI エージェント向け金融インフラ「Circle Agent Stack」を発表

## 要約
Circle は 2026-05-11、Agent Wallets・Agent Marketplace・Nanopayments・Circle CLI・Circle Skills の 5 コンポーネントからなる「Circle Agent Stack」をリリース。Nanopayments は最小 $0.000001 のガス不要 USDC 転送をマシン速度で実現。CEO Allaire は「AI エージェント自身が顧客」と位置付ける初の本格的エージェント向け金融スタックと説明。即日 agents.circle.com で提供開始。

## なぜ KYA に関係あるか
Agent Wallets の「policy-controlled」設計と spending guardrails はノンカストディ型 KYA との競合・補完関係を直接示す。エージェント ID 管理の標準化に Circle が乗り出したことで Track A の競合地図が更新される。
