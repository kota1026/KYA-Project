# Privado ID「Know Your Agent」— DID + VC によるエージェント識別フレームワーク

## 要約
Privado ID が Agent JS SDK を提供し、エージェントが自己の DID を生成・暗号署名で制御を証明するフレームワーク「Deep Trust」を発表。鍵管理は外部 KMS（HSM・MPC・Lit Protocol 統合）に委譲。エージェントが発行した VC をクロスアプリ環境で再利用可能な設計。プライバシー保護型 VC がレガシーシステムと相互運用できる構造をミドルウェア層で提供。

## なぜ KYA に関係あるか
KYA プロジェクトの直接競合候補。DID + VC ベースのエージェントID実装は自社ノンカストディ型設計との差別化分析が必要。Track A 競合プロファイルおよび Track C 技術評価の両方に関連。
