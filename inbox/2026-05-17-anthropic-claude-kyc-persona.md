# Anthropic — Claude ユーザーに政府発行 ID + セルフィー要求開始

## 要約
Anthropic が Persona Identities と連携し、Claude の一部機能にアクセスする際に政府発行 ID（パスポート・運転免許証・国民 ID）とライブセルフィーを要求する ID 確認をロールアウト（2026-04-16）。データは Anthropic ではなく Persona が保管し、モデル学習には不使用。ChatGPT・Gemini は同等の本人確認を未実施であり競合差異が生じている。ユーザーコミュニティからのプライバシー懸念が表出。

## なぜ KYA に関係あるか
「エージェントを動かす人間」の本人確認を LLM プラットフォーム層が直接担う先例となりうる。KYH（Know Your Human）アプローチの実装例として Track B の人間責任バインディング論点に組み込み要。PYMNTS の KYH 記事と合わせて参照。
