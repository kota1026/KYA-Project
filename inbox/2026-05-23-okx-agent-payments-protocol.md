# OKX、独自 Agent Payments Protocol (APP) を発表 — ビジネスサイクル全体(見積→決済→紛争)を AI で自動化

## 要約
OKX が独自の Agent Payments Protocol (APP) を発表(2026-05-01)。x402(Coinbase)・MPP(Stripe)に続く第三のエージェント決済プロトコル。見積もり・決済・紛争処理を含むビジネスサイクル全体をボットが操作可能な設計が特徴。複数チェーン対応予定、X Layer の Payment SDK でゼロ〜低ガス取引を実現。

## なぜ KYA に関係あるか
x402・AP2・MPP・OKX APP と複数プロトコルが競合する中、KYA の委任 Credential がどのプロトコルに対応すべきかが重要設計課題となる。OKX の紛争処理機能は KYA の Accountability 要件(誰が責任を持つか)と直接関係する。Track A の競合マップに追加必須。
