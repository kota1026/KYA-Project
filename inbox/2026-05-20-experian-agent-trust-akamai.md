# Experian が Agent Trust エコシステムに Akamai を追加（Visa・Cloudflare・Skyfire と並列）

## 要約
Experian Agent Trust フレームワーク（4/30 発表）のパートナーエコシステムに Akamai が 5/15 参加。既存パートナーは Visa・Cloudflare・Skyfire。Agent Trust は「Human-to-Agent Binding」を核に identity・consent・fraud risk を Agent Trust Token として提供。KYAPay で agent intent を宣言しトークナイズド支払認証を実現する設計。

## なぜ KYA に関係あるか
Experian が大手インフラ企業と組んで KYA 類似ソリューションをエコシステム型で展開。Skyfire との連携は競合プレイヤー間の連合体形成を示唆。Track A の「エコシステムプレイヤーマップ」更新が必要。
