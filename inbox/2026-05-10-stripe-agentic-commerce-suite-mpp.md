# Stripe Agentic Commerce Suite 発表 + Machine Payments Protocol (MPP) ローンチ

## 要約
Stripe がエージェント向け決済スイートを発表。Shared Payment Token (SPT) を新プリミティブとして導入——エージェントが実際の決済情報を持たずに支払いを起動可能。Link ウォレットをエージェント対応に拡張。MPP（Machine Payments Protocol）を Tempo と共同仕様化しオープン化。Mastercard Agent Pay・Visa Intelligent Commerce・BNPL（Affirm/Klarna）への SPT 拡張を予定。

## なぜ KYA に関係あるか
Stripe は Track A の主要競合プレイヤー。SPT の設計が KYA の「エージェント認証なしに決済できるか」問題への回答を示す。
