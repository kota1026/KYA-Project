# Stripe Sessions 2026 — Link ウォレットのエージェント対応 + Agentic Commerce Suite

## 要約
Stripe Sessions 2026（2026年4月）で288の新機能を発表。Link ウォレットを AIエージェントが使える OAuth フローで拡張し、エージェントが支払いリクエストを生成→ユーザー承認→実行するモデルを採用。Google（UCP経由のGeminiアプリ内購入）・Meta（Facebook広告内ネイティブ決済）と統合。Affirm BNPL との連携も発表。Google とのパートナーで同社を AP2 エコシステムに取り込む形。
