# IACR ePrint 2026/790：レガシーシステム PQC 移行のリスクベース・フレームワーク

## 要約（3-5文）
"Towards a Field-Informed Risk-Based Framework for PQC Migration in Legacy Systems"。暗号的アジリティが低いレガシー基盤のPQC移行に特化したリスク分類フレームワークを提案。規制制約・インフラ老朽化・移行コストの3軸で優先順位付け手法を提示。実務寄りのガイドラインとして産業界向けの参照価値あり。

## なぜ KYA に関係あるか
KYAのIdentity・署名基盤の長期設計においてPQCへの移行タイムラインは検討要素（QSBはプライベート資産として温存済み）。直接的優先度は低いが Track C PQC章の参考文献として記録。
