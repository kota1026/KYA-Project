# Biometric Update「アジェンティック AI が金融セクターを "継続的 ID" へ押し進める」

## 要約

金融機関が KYC 中心の静的 ID 検証から「継続的 ID 検証(continuous identity verification)」へ移行しつつあるという業界トレンド記事。エージェントは人間と異なり時刻・場所・デバイスの変動なしに継続稼働するため、行動バイオメトリクスに基づく異常検知も機能しにくい。業界回答は、生体認証・継続的モニタリング・デバイスインテリジェンスの組み合わせとされているが、エージェント固有のソリューションはまだ成熟していないと指摘。

## なぜ KYA に関係あるか

KYA は「誰がエージェントを認可したか」の初期検証だけでなく、継続的な権限状態の監視も含むべきという設計要件を示唆する。Track B での規制要件定義(continuous vs. point-in-time)と、Track C での技術要件(VC revocation, session scoping)に関連する。
