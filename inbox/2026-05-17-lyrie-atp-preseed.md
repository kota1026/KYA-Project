# Lyrie.ai — $2M プレシード調達 + Agent Trust Protocol (ATP) 公開

## 要約
OTT Cybersecurity が開発する Lyrie.ai が $2M プレシードを完了しステルスから離脱。AI エージェント向けオープン暗号標準「Agent Trust Protocol (ATP)」を公開。ATP は Identity・Scope・Attestation・Delegation・Revocation の 5 軸をカバーし、IETF 提出予定かつ MIT ライセンスで参照実装を GitHub 公開。同社は Anthropic Cyber Verification Program (CVP) の第 1 期採択企業にも選定された。

## なぜ KYA に関係あるか
ATP は KYA の技術的コアである「エージェント身元証明の暗号標準化」をまさに実装しようとしている。IETF 提出により業界標準になる可能性があり、Track A の競合地図に新規プレーヤーとして追加必須。Track C の標準化動向にも直接影響。
