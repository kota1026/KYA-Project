# Oasis Security、$120M Series B 調達 — NHI・エージェントアクセス管理プラットフォーム

## 要約
Oasis Security が Craft Ventures リードで Sequoia・Accel 参加の Series B $120M を調達(累計 $195M)。非人間 ID(NHI: サービスアカウント、API キー、AI エージェント)のライフサイクル管理と「Agentic Access Management (AAM)」が主力製品。Just-in-time アクセス・常設権限ゼロ・シングルポリシー制御が特徴。Fortune 500 顧客の ARR が前年比 5 倍成長。

## なぜ KYA に関係あるか
NHI/エージェント ID セキュリティ市場で有力プレイヤー。KYA が構築しようとする「エージェント ID + スコープ制御」と機能が重複・競合する可能性。Track A のプロファイル作成が必要。
