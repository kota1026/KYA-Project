# arXiv:2604.15367 — SoK: Security of Autonomous LLM Agents in Agentic Commerce

## 要約（3-5 文）
自律型 LLM エージェントによるアジェンティックコマースのセキュリティを体系化した SoK（Systematization of Knowledge）論文。攻撃ベクタをプロンプトインジェクション・エージェントスプーフィング・不正権限昇格などに分類。ERC-8004 の実装が提供するエージェント ID・支出上限・失効機構を評価し、実用ガードレールの不足を指摘する。既存プロトコル（x402 等）の KYA ギャップを一覧化している。

## なぜ KYA に関係あるか
ERC-8004 をはじめとするエージェント ID 標準の現状ギャップが網羅的に整理されており、KYA テクニカルスタック設計（Track C + Track A クロスリファレンス）の基礎資料として高価値。
