# Sumsub — AI Agent Verification（Agent-to-Human Binding）発表

## 要約
2026-01-29、Sumsub がKYAフレームワーク下でAI Agent Verification製品をリリース。エージェントの活動を検証済み人間IDにバインドし「エージェント-人間の説明責任チェーン」を確立。デバイスインテリジェンス・マルアカウント検知・ライブネス検出を組み合わせ。同社Identity Fraud Report 2025-2026によれば2025年に多段階協調攻撃が前年比180%増。

## なぜ KYA に関係あるか
KYAの直接競合製品。「エージェントIDを人間のKYC済みIDにバインドする」設計はKYAの非カストディアーキテクチャ検討の対比事例として最重要。competitor-researcherで深堀り必須。
