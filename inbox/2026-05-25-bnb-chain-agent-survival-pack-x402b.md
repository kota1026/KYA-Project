# BNB Chain、Agent Survival Pack でオンチェーン AI 決済エコシステムを構築

## 要約

BNB Chain が 2026-05-25 に "Agent Survival Pack" を発表。6 社のパートナー(Pieverse、Bankr、WorldClaw、B.AI、AEON、Alt AI)と連携し、AI エージェントが自律的に運営コストをオンチェーン決済できるインフラを整備した。Pieverse は x402 HTTP 支払い標準を BNB Chain 向けに拡張した "x402b" を採用。B.AI は ERC-8004 によるオンチェーン ID とオンチェーン決済を統合した金融レイヤーを提供する。AEON は東南アジアの Visa/Mastercard 物理加盟店での QR 決済を可能にする。

## なぜ KYA に関係あるか

x402b は x402 の BNB Chain 拡張であり、Track C の標準化動向に直結する。B.AI による ERC-8004 オンチェーン ID は KYA の Agent Identity 具体実装例として Track A / Track C 両方に資料価値がある。競合マッピング(Track A)において BNB Chain エコシステム全体が新たな勢力として浮上。
