# Consensus Miami 2026: Trust Wallet と Mesh が AI エージェント向けウォレット再設計を発表

## 要約
Consensus Miami 2026 にて Trust Wallet CEO Felix Fan と Mesh CTO Arjun Mukherjee が AI エージェント向けウォレット設計の方向性を公表。Trust Wallet は EIP-8004(エージェント ID・スコアリング標準)を採用したエージェントツールキットを公開。ウォレットを「新しいブラウザ」と位置づけ、エージェントがスワップ・ブリッジ・イールド管理を自律実行。Mesh は AI 駆動型決済とプログラマブルマネーで企業財務の自動化を訴求。セキュリティ上の懸念として、AI 統合 API・プラグイン・ミドルウェアの脆弱性が 2026-04-13 に指摘されている(CryptoRank レポート)。

## なぜ KYA に関係あるか
EIP-8004 は KYA における agent identity / scoring の標準候補。Trust Wallet の実装はノンカストディ型に近く、Track B 規制適合との整合を確認すべき。AI 統合ミドルウェアの脆弱性はエージェント authentication の弱点として Track C に記録価値あり。
