# Stripe Agentic Commerce Suite 発表 — OpenAI と共同で ACP を策定

## 要約
Stripe Sessions 2026でAgentic Commerce Suiteを発表。OpenAIとのAgentic Commerce Protocol(ACP)を共同策定し、ChatGPTに「Instant Checkout」を提供。Mastercard Agent PayとVisa Intelligent Commerceにも対応。Google GeminiアプリへはUniversal Commerce Protocol(UCP)経由で接続。288の新製品・機能を発表し、フィアット寄りエージェント決済の統合ソリューションを一気に拡充。

## なぜKYAに関係あるか
Stripeが既存カード決済インフラをエージェント決済に拡張する戦略は、KYAが対応すべき「フィアット側」の主要スタックを代表する。ACPとAP2・x402との標準競合構図はKYA設計に直接影響。Track A競合プロファイル要更新。
