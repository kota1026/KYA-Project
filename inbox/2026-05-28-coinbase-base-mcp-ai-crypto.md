# Coinbase Base、MCP ゲートウェイで Claude/ChatGPT から直接オンチェーン取引を可能に

## 要約
CoinbaseのBase L2が2026年5月26日にBase MCPをローンチ。AIエージェント（Claude、ChatGPT等）がBase上のスマートウォレットに接続し、USDC送金・DeFiスワップ・x402マイクロペイメントを自然言語指示で実行できる。秘密鍵はユーザー側ウォレットに留まり、エージェントの操作は都度ユーザー承認を要求する設計。x402対応APIへのUSDC支払いもエージェントが代行可能。

## なぜ KYA に関係あるか
MCPプロトコルがエージェントと金融インフラを繋ぐ実装層として定着しつつある。Baseが「x402 + MCP + スマートウォレット」の三位一体構成を実装したことで、エージェントの支払い委任モデルにおける標準スタックが見えてきた。Track C の「プロトコル競争」欄（x402 vs AP2 vs MPP）に追加し、Coinbase側の技術選択を記録する。
