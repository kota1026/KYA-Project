# Stellar x402: Unlocking Payments for the New Agent Economy

## 要約（3-5文）
Stellar Development Foundation が x402 プロトコルの Stellar ネットワーク統合を公式発表。Stellar の決済特性（5秒以内決済・手数料0.00001ドル・USDC/PYUSD/USDYネイティブ対応・99.99%稼働率）がx402の同期HTTPリクエスト内精算要件に適合すると主張。またMPP（Machine Payments Protocol）も Stellar 上で稼働させ「x402 = リクエスト単位課金、MPP = セッション単位ストリーム決済」という二層構成を提示。2026年5月時点でx402は月間289万件・調整後取引高119万ドルをBase/Solanaで処理中（Stellar 分は追加）。

## なぜ KYA に関係あるか
x402 決済レイヤーの多チェーン化は KYA システムがウォレット層を設計する際の「どのチェーン上で精算するか」という技術選定に影響する。Stellar の低手数料・高稼働率は小口エージェント決済に適しており、Track C の x402 エコシステムマップに記録必要。
