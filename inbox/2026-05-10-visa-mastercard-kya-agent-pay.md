# Visa Intelligent Commerce + Mastercard Agent Suite: カードネットワークの KYA アプローチ

## 要約
Visa が AI-Ready Cards（エージェント・デバイス・マーチャントにスコープされたトークン化認証）を展開。100 社超のパートナーが参画。Mastercard は Q2 2026 に Agent Suite ローンチ——Agentic Token による暗号署名認証を採用。2025年12月に Skyfire が KYAPay + Visa Intelligent Commerce の統合で初の本番 KYA 決済を実証済み。

## なぜ KYA に関係あるか
Visa/MC 自身が KYA フレームワークを実装しつつある。Skyfire との統合実績は Track A の競合動向として重要。
