# Amazon Bedrock AgentCore Payments (Preview) — AWS × Coinbase × Stripe

## 要約
2026-05-07、AWSがAmazon Bedrock AgentCore Paymentsをプレビュー公開。Coinbase（x402プロトコル / USDCウォレット）とStripe（Privyウォレット）をバックエンドに、AIエージェントが自律的にAPIやMCPサーバーへのマイクロ決済を実行できるマネージドインフラ。セッション単位の支出上限・監査ログ付き。USDC決済はBase / Solana上で即時決済。現在US・EU・APACの4リージョンで利用可能（プレビュー）。

## なぜ KYA に関係あるか
x402をベースにした「エージェント向けウォレット + 支出ガバナンス」が大手クラウドに組み込まれた。KYAの文脈では「誰がエージェントに資金を持たせるか・どう制限するか」という権限管理モデルの参照実装になる。競合ポジション（Track A）および事業モデル（Track D）に直結。
