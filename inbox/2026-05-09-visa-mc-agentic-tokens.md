# Visa Intelligent Commerce + Mastercard Agent Pay — カードネットワークのエージェント対応

## 要約
VisaはIntelligent Commerceフレームワークを拡大：スコープ付きトークン化クレデンシャルをAIエージェントに発行、Anthropic/OpenAI/Microsoftと統合。2026年ホリデーシーズンまでに数百万人利用を目標。MastercardはAgent Payを発表：Agentic Tokens（支出上限・取引先ホワイトリスト・カテゴリ制限付き）。双方がStripe SPTとも連携。

## なぜ KYA に関係あるか
既存カードレールへのエージェント統合が加速。KYAの競合環境分析（Track A）においてカードネットワーク勢の動向は必須。トークン化アプローチとDID/VC型KYAの棲み分けを評価する必要あり。
