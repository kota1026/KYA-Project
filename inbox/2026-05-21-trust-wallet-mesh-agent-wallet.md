# Consensus Miami 2026 — Trust Wallet & Mesh、エージェントウォレット再構築戦略を表明

## 要約（3-5 文）
Trust Wallet CEO Felix Fan と Mesh CTO Arjun Mukherjee が Consensus Miami でエージェント対応ウォレット戦略を公表。Trust Wallet は EIP-8004 でオンチェーン ID を付与しつつ、コンシューマー向けには人間が鍵を保持するコパイロット型を維持する二層構造を採用。Mesh の「Smart Funding」はクロスチェーン決済経路を自動選択し、エージェントのコールドスタート問題（資金調達なしでは動けない）を解消する。両社とも AI ラボによる自前ウォレット参入の可能性を示唆した。

## なぜ KYA に関係あるか
EIP-8004 採用事例とノンカストディ型の実装事例として Track A に追加価値あり。「コールドスタート問題」はエージェント ID と資金調達を接続する KYA 実務課題と直結する。
