# SecureAuth、業界初の公開 Agent Trust Registry を開設 — エージェント ID ポータビリティを提供

## 要約
SecureAuth が業界初となる公開型 Agent Trust Registry を 2026-04-29 に開設。企業は AI エージェントを登録・管理し、信頼スコアや権限スコープを外部に開示可能。AI エージェントが「エスカレーションするセキュリティ脅威」をもたらすと位置付け、レジストリによる可視化で対応。登録件数や利用規模は未確認。

## なぜ KYA に関係あるか
KYA フレームワークの中核要件「エージェント登録」を実装した具体サービス。競合他社がどのようなレジストリ設計を採用しているかを把握する上で参照価値が高い。Track A・C 両方に関連。
