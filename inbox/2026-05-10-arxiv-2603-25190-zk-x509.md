# zk-X509: Legacy PKI から ZKP でオンチェーン ID を生成（arXiv:2603.25190）

## 要約
既存 X.509 証明書を RISC-V zkVM 上で証明し、秘密鍵や個人識別子を開示せずオンチェーン ID を確立するシステム。レガシー PKI インフラとパブリックチェーンをプライバシー保護した形でブリッジ。エージェントが既存 TLS 証明書チェーンを活用して自律的に身元証明できる可能性がある。

## なぜ KYA に関係あるか
KYA インフラが新規 DID に頼らず既存 PKI を活用できる場合、導入摩擦が大幅に低下。zkTLS との組み合わせ可能性を評価する価値あり。
