# RSAC 2026：Cisco・CrowdStrike ら 5 社がエージェント ID フレームワーク発表、IAM 成熟度モデルも登場

## 要約
RSAC 2026 でエージェント ID が主要テーマとして浮上。Cisco、CrowdStrike、Palo Alto Networks、Microsoft、Cato Networks の 5 社がエージェント ID フレームワークを発表。調査では現行 IAM システムがエージェント ID を管理できると「高い確信」を持つセキュリティリーダーはわずか 18%。VentureBeat が 6 段階のエージェント ID 成熟度モデルを提示。

## なぜ KYA に関係あるか
大手セキュリティベンダーが「エージェント ID」をエンタープライズ必須課題として本格対応し始めた。KYA の市場環境（IAM ベンダーが KYA 類似機能を組み込む競合リスク）を理解する上で重要。
